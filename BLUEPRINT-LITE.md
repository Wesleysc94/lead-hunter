# Lead Hunter - Blueprint Lite

## Positioning

Technical automation system built to transform raw local-business discovery into qualified leads, reusable contact context, and operational visibility.

## Core Direction

- automation-first workflow with a human-in-the-loop outreach model;
- emphasis on qualification quality rather than raw scraping volume;
- local execution for sensitive credentials and private exports;
- public dashboard used as a technical showcase, not as a credentialed admin panel.

## Main Flow

1. Search businesses by city and category
2. Detect current digital maturity
3. Enrich data with Instagram signals
4. Score leads
5. Draft assisted outreach messages
6. Export qualified leads
7. Surface results in a dashboard

## Inputs

- location and category targets
- API credentials
- Google Sheets target
- messaging and notification settings

## Outputs

- qualified leads
- score bands
- message suggestions
- dashboard metrics
- CSV, JSON, HTML exports

## Why This Works

- shows applied automation, not only scripting;
- combines data collection, qualification, and operational visibility;
- demonstrates system thinking, integration discipline, and technical productization.
